To link to another topic or external site inline, you use the xref
element.

As a general best practice, peopl says to avoid inline links if possible, but include an
inline link it would be odd not to include the link. Or include the inline link if there's no
concern about the target not being included in the output. 

<p class="p">The topic title will automatically be used as the anchor text for the link. </p>

    <pre class="pre codeblock">   &lt;xref href="sample.dita"/&gt;</pre>

  <p>If you want to manually specify the anchor text, code it like this:
    <pre class="pre codeblock">   &lt;xref href="sample.dita"&gt;Sample&lt;/xref&gt;</pre>

If you're linking to an external source, you need to add the scope and
format attributes to your link:

   <xref href="http://example.org" scope="external" outputclass="external" format="html">Example.org</xref>
